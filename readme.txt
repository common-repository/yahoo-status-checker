Yahoo! Status Checker by Techya.Check any yahoo id to see if he is online or not.

== Description ==

Yahoo! Status Checker is a simple plugin which you can use on your wordpress websites as an new feature or advantage for your users with the ability to check if the particular yahoo user is online or offline.

== Installation ==

1. Upload `yahoo_status` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets section
4. Drag the 'Yahoo! Status Checker' to the sidebar

Its done! Now,you have your own yahoo status checker on your sidebar.

== Frequently Asked Questions ==

= What does this plugin do? =

It checks if a provided yahoo user is online or offline.
